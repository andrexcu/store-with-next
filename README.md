### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/andrexcu/store-with-next.git
```

### Install packages

```shell
npm i
```

### Setup .env file


```js
NEXT_PUBLIC_API_URL=
```


### Start the app

```shell
npm run dev
```
